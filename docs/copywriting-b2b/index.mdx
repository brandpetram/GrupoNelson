---
title: Copywriting B2B
description: Sistema completo de escritura comercial para sitios web de empresas B2B industriales
---

Sistema de producción de copy comercial B2B para clientes de Brandpetram. Incluye reglas de escritura, vocabularios por sector, plantillas reutilizables y guías del proceso.

Este sistema tiene dos tipos de documentos:
- **Genéricos** — reglas, vocabularios y plantillas que se reutilizan tal cual entre proyectos
- **Específicos del cliente** — briefs, buyer personas y temarios que viven dentro de cada proyecto en `docs/copy/`
