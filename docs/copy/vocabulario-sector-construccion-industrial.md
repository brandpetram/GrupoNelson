# Vocabulario de Sector: Construcción Industrial

Términos específicos del sector de construcción industrial (design-build, build-to-suit, construcción de naves). Estas son las palabras que un director de construcción, un project manager, un ingeniero estructural o un cliente que evalúa constructoras usa internamente.

Se complementa con `vocabulario-generico-b2b.md` (obligatorio leer primero).

---

## Modelos de Construcción

| Término | Qué significa | Cuándo usar |
|---------|---------------|-------------|
| Design-build | Diseño y construcción bajo un solo contrato y un solo responsable | Modelo principal de Baumex |
| Build-to-suit | Construcción a la medida exacta de las especificaciones del cliente | Servicio diferenciador |
| Turnkey / Llave en mano | Entrega lista para operar — el cliente entra y arranca | Propuesta de valor |
| General contractor | Contratista principal que coordina toda la obra | Rol de Baumex |
| In-house construction | Constructora propia del desarrollador (no subcontratada) | Ventaja vs. competencia |
| EPC (Engineering, Procurement, Construction) | Modelo integral de ingeniería, procura y construcción | Proyectos complejos |

## Fases de Construcción

| Término | Qué significa | Cuándo usar |
|---------|---------------|-------------|
| Proyecto ejecutivo | Planos y especificaciones finales para construir | Fase de diseño |
| Terracerías / Earthworks | Movimiento de tierras, nivelación, compactación | Primera fase de obra |
| Cimentación / Foundation | Bases estructurales del edificio (zapatas, pilas, dados) | Fase estructural |
| Pilas de concreto armado | Cimentaciones profundas cilíndricas para suelos inestables | Proyectos especiales en zona sísmica |
| Zapatas | Cimentaciones superficiales — bases de concreto | Cimentación estándar |
| Estructura metálica | Esqueleto de acero del edificio (columnas, vigas, armaduras) | Fase estructural |
| Mampostería / Masonry | Muros de block de concreto | Fase de muros |
| Tilt-up | Sistema con paneles de concreto levantados con grúa | Tipo de construcción |
| Pre-engineered metal building (PEMB) | Estructura metálica prefabricada en planta | Tipo de construcción |
| Techumbre / Roofing | Instalación de cubierta (lámina, paneles, TPO) | Fase de cerramiento |
| Fachada / Sheet metal walls | Muros exteriores metálicos con aislante | Fase de cerramiento |
| Tablaroca / Drywall / Sheet rock | Muros interiores de yeso | Fase de interiores |
| Acabados / Finishes | Pisos, pintura, sanitarios, cancelería | Fase final |
| Punch list | Lista de detalles pendientes antes de entrega final | Cierre de proyecto |

## Instalaciones Especializadas

| Término | Qué significa | Cuándo usar |
|---------|---------------|-------------|
| Instalación eléctrica | Tableros, cableado, conduit, luminarias, subestaciones | Sistemas del edificio |
| Instalación mecánica | HVAC, ductos, plomería, gas | Sistemas del edificio |
| HVAC | Heating, Ventilation and Air Conditioning | Clima artificial |
| Chiller | Equipo de enfriamiento industrial de gran capacidad | Sistemas de A/C |
| Manejadora de aire / Air handling unit | Equipo que inyecta aire acondicionado | HVAC industrial |
| Fire suppression / Sistema contraincendio | Rociadores, tanques, bombas, tubería | Seguridad — obligatorio |
| Sprinkler system | Sistema de rociadores automáticos | Contraincendio |
| ESFR (Early Suppression Fast Response) | Tipo avanzado de sprinkler para naves altas | Specs técnicos |
| Subestación eléctrica | Transformación de energía para el edificio o parque | Infraestructura eléctrica |
| Capacidad instalada | KVA o MW disponibles | Specs eléctricas |
| Torres de enfriamiento | Equipos para disipar calor del sistema HVAC | Instalaciones mecánicas |

## Gestión de Proyecto (Project Management)

| Término | Qué significa | Cuándo usar |
|---------|---------------|-------------|
| Ruta crítica / Critical path | Secuencia de tareas donde un retraso retrasa todo el proyecto | Control de tiempos |
| Gantt chart / Cronograma | Diagrama de barras con todas las actividades y sus fechas | Planeación |
| Submittals | Especificaciones de materiales enviadas al cliente para aprobación antes de instalar | Control de calidad |
| Long lead items | Equipos con tiempos de entrega largos (transformadores, chillers, bombas) | Planeación de procura |
| Minutas de reunión | Registro formal de acuerdos y responsables por reunión | Control administrativo |
| Weekly report | Reporte semanal de avance de obra | Seguimiento |
| Horas Hombre Trabajadas (HHT) | Métrica estándar de volumen de trabajo y seguridad | Seguridad y escala |
| ERP Construction Management | Software de gestión centralizada de obra | Herramientas |
| Pre-construction meeting | Reunión antes de iniciar obra para alinear a todos los involucrados | Proceso |
| Quality testing | Ensayos de calidad por laboratorio externo (compactación, concreto) | Control de calidad |

## Seguridad Industrial (HSE)

| Término | Qué significa | Cuándo usar |
|---------|---------------|-------------|
| HSE | Health, Safety & Environment — salud, seguridad y medio ambiente | Marco general de seguridad |
| Daily work permit | Permiso de trabajo diario con matriz de riesgos antes de iniciar | Control diario |
| Matriz de riesgo | Evaluación de Impacto × Probabilidad para cada actividad | Herramienta de seguridad |
| PPE / EPP | Personal Protective Equipment / Equipo de Protección Personal | Seguridad básica |
| DC-3 | Certificación mexicana de capacitación laboral | Compliance |
| NOM-009-STPS-2011 | Norma para trabajo en alturas | Permisos especializados |
| NOM-027-STPS-2008 | Norma para trabajo con fuego (soldadura, oxicorte) | Permisos especializados |
| Vigía de incendios | Persona designada para vigilar riesgo de fuego durante soldadura | Protocolo de seguridad |
| Safety meeting | Reunión de seguridad con todo el personal antes del turno | Práctica diaria |
| Cero accidentes fatales | Meta de seguridad — siempre con HHT para dar contexto | Prueba de cultura de seguridad |
| Scorecard de seguridad | Tablero con métricas de incidentes, días sin accidentes, récords | Transparencia |
| IMSS / SUA | Seguro social mexicano / Sistema Único de Autodeterminación | Compliance laboral |

## Certificaciones y Normativas de Construcción

| Término | Qué significa | Cuándo usar |
|---------|---------------|-------------|
| LEED | Leadership in Energy and Environmental Design — niveles: Certified, Silver, Gold, Platinum | Edificios verdes |
| FM Global / Factory Mutual | Certificadora internacional — reduce primas de seguro significativamente | Muros, cubierta, estructura, contraincendio |
| NFPA | National Fire Protection Association | Sistemas contraincendio |
| ISN / ISNetworld | Plataforma global de gestión de contratistas — calificación "Recomendado" | Acceso a cadena de proveedores de Fortune 500 |
| IBC | International Building Code | Códigos de construcción internacional |
| NOM | Norma Oficial Mexicana — regulaciones técnicas obligatorias | Compliance local |
| NMX | Norma Mexicana — regulaciones técnicas voluntarias (acero, concreto) | Materiales |
| STPS | Secretaría del Trabajo — normas de seguridad laboral | Compliance laboral |
| ESG | Environmental, Social, Governance | Reportes corporativos del cliente |
| BRB (Buckling-Restrained Braces) | Disipadores de energía antisísmica — tecnología de Japón/CDMX | Zona sísmica, proyectos especiales |
| TPO (Thermoplastic Polyolefin) | Sistema de cubierta de alta gama — garantía 20 años impermeabilización | Specs de techo premium |

## Términos Técnicos Específicos

| Término | Qué significa | Cuándo usar |
|---------|---------------|-------------|
| Clear height / Altura libre | Distancia del piso al obstáculo más bajo del techo | Specs de naves — criterio #1 |
| Slab / Firme | Piso de concreto de la nave | Specs de piso |
| Floor load capacity / Carga de piso | Peso que soporta el piso (kg/m² o PSI) | Manufactura pesada |
| Column spacing | Distancia entre columnas interiores | Layout de producción |
| Dock / Andén | Punto de carga/descarga de camiones | Logística |
| Drive-in door | Puerta a nivel de piso para vehículos | Acceso |
| Zona sísmica | Clasificación de riesgo sísmico del sitio (zona 4 = alta) | Diseño estructural |
| Mecánica de suelos | Estudio de las condiciones del terreno antes de construir | Cimentación |
| Concreto premezclado | Concreto fabricado en planta y transportado en revolvedora | Materiales |
| Colado | Vertido de concreto en moldes o cimbra | Proceso constructivo |
| Placa estructural | Acero grueso para aplicaciones de alta resistencia (vs. lámina) | Proyectos especiales |
| Anclas embebidas | Piezas metálicas fijadas en concreto para montar equipos | Precisión milimétrica |
