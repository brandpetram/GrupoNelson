# Qué es lo que estamos haciendo

Crear un sitio web comercial para **Grupo Nelson** que comunique autoridad en los temas comerciales a los que Grupo Nelson se dedica desde hace **65 años**: la **proveeduría de naves y parques industriales en Mexicali**.

Grupo Nelson tiene una serie de espacios industriales disponibles, llamados **Naves Industriales**.
