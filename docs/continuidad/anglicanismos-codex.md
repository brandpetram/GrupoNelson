# Anglicanismos en la version en espanol del sitio

Este documento registra los anglicismos detectados en la version en espanol del sitio de Grupo Nelson y define cuales conviene pasar al espanol. La regla editorial propuesta es simple:

- Traducir anglicismos de copy comercial cuando existe una forma clara en espanol.
- Mantener siglas, certificaciones, normas, unidades y nombres de categorias reconocidas internacionalmente.
- En terminos tecnicos usados por SEO o por compradores industriales, usar espanol en el texto principal y, cuando sea necesario, conservar el termino ingles solo como apoyo entre parentesis la primera vez.

## Nota de aplicacion

Cuando una palabra este dentro de una cita, nombre oficial, ruta, slug, codigo, import, nombre de componente, clase CSS o nombre de archivo, no debe reemplazarse automaticamente. Esta guia aplica al texto visible del sitio: titulos, parrafos, descripciones SEO, labels, tablas, fichas tecnicas, articulos y noticias.

Para terminos con valor SEO, la formula recomendada es:

- Primera mencion: `altura libre (clear height)`.
- Menciones posteriores: `altura libre`.

Aplicar la misma logica a `almacenamiento en frio (cold storage)`, `construccion a la medida (build-to-suit)` y `relocalizacion cercana (nearshoring)`.
